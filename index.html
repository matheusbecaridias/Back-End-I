🚀 Desenvolvimento Back-End

Construindo aplicações robustas e escaláveis

Sobre a Disciplina

A disciplina de Desenvolvimento Back-End prepara os estudantes para criar e manter aplicações de servidor confiáveis, escaláveis e seguras. Você aprenderá os fundamentos e as melhores práticas para desenvolvimento de sistemas backend modernos.

Objetivos da Disciplina

  • Compreender os princípios fundamentais da arquitetura de servidor
  • Desenvolver APIs RESTful usando JavaScript/Node.js
  • Trabalhar com bancos de dados relacionais e não-relacionais
  • Implementar autenticação e autorização seguras
  • Aplicar padrões de design e arquitetura de software
  • Criar testes automatizados para garantir qualidade do código

Tópicos Principais

Fundamentos

  • JavaScript avançado
  • Node.js
  • Express.js
  • HTTP e protocolos

Banco de Dados

  • SQL e PostgreSQL
  • MongoDB
  • Queries otimizadas
  • Modelagem de dados

Segurança

  • Autenticação JWT
  • Validação de dados
  • Proteção contra ataques
  • Variáveis de ambiente

Ferramentas

  • Git e GitHub
  • Docker
  • Testing (Jest, Mocha)
  • Deploy e CI/CD

Metodologia

A disciplina utiliza uma abordagem prática e baseada em projetos. Os estudantes trabalharão em desafios reais, construindo aplicações completas do zero, desde o planejamento até o deploy em produção.

Requisitos

  • Conhecimento básico em programação JavaScript
  • Familiaridade com conceitos de Internet e HTTP
  • Git instalado e configurado
  • Editor de código (VS Code recomendado)

Tecnologias Utilizadas

Linguagens: JavaScript, SQL, NoSQL
Frameworks: Node.js, Express.js
Bancos de Dados: PostgreSQL, MongoDB
Ferramentas: Git, Docker, Postman, Jest
Concepts: REST APIs, Microserviços, Cloud Computing
